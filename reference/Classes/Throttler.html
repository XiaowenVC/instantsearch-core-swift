Throttler

@objc public class Throttler: NSObject, Caller

Ensures a maximum throughput on a stream of calls. The throughput is expressed as a minimum delay between calls, via the delay property.

Properties
  • delay

    Minimum delay between two calls.

    Declaration

    Swift

    @objc public var delay: TimeInterval
  • fireInitialCall

    Whether to fire immediately the initial call in a series. Default = true.

    A series begins when a call is enqueued, and ends when the delay expires without any call being fired. When this property is true, the initial call in a series will be fired immediately; subsequent calls will be enqueued for deferred firing. When it is false, all calls are enqueued, including the initial call.

    Declaration

    Swift

    @objc public var fireInitialCall: Bool = true
