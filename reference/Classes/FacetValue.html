FacetValue

@objc public class FacetValue: NSObject

A value of a given facet, together with its number of occurrences. This class is mainly useful when an ordered list of facet values has to be presented to the user.

Properties
  • value

    Value of the facet.

    Declaration

    Swift

    @objc public let value: String
  • count

    Number of occurrences of the value.

    Note

    If SearchResults.exhaustiveFacetsCount is true, it may be approximate.
