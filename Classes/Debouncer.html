Debouncer

@objc public class Debouncer: NSObject

Merges multiple calls into a single one.

The debouncer works by delaying every call by a given delay; if another call happens before the delay is expired, the timer is reset and the first call is ignored. When the delay expires, the last call is fired.

    Amount of time by which calls will be delayed before being fired.
