InstantSearch Core
for Swift and Objective-C

Build rich search experiences

This is the InstantSearch Core library for Swift and Objective-C, a companion to Algolia’s Swift API Client, using Algolia’s Search API. It provides a set of utilities to ease the development of rich search experiences.

Get Started

Mobile-ready

InstantSearch Core works out of the box on iOS. Of course, macOS, tvOS and watchOS are supported as well!

Member of a larger family

Looking for something similar on Android? or for the web? Have a look at InstantSearch Android or instantsearch.js.
